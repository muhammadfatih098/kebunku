Kebijakan Privasi Kebunku

Terakhir diperbarui: April 2026

Kebunku adalah aplikasi panduan perawatan tanaman hias. Kami berkomitmen untuk melindungi privasi pengguna.

Data yang Dikumpulkan

Kebunku tidak mengumpulkan, menyimpan, atau mengirimkan data pribadi ke server manapun. Semua data tersimpan secara lokal di perangkat pengguna.

Penyimpanan Lokal

Aplikasi menyimpan data berikut hanya di perangkat Anda:

  • Daftar tanaman yang dibookmark
  • Preferensi pengaturan notifikasi

Notifikasi

Aplikasi meminta izin notifikasi untuk mengirimkan pengingat jadwal penyiraman tanaman. Izin ini sepenuhnya opsional dan dapat dinonaktifkan kapan saja melalui pengaturan perangkat.

Koneksi Internet

Kebunku tidak memerlukan koneksi internet. Tidak ada data yang dikirim ke server eksternal.

Pihak Ketiga

Kebunku tidak menggunakan layanan analitik, iklan, atau pihak ketiga lainnya yang dapat mengumpulkan data pengguna.

Anak-anak

Aplikasi ini tidak ditujukan untuk anak-anak di bawah usia 13 tahun dan tidak mengumpulkan data dari mereka.
